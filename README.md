## 🚀 Featured Project

### 🧠 AI Resume & Job Match Analyzer
Python | NLP | SentenceTransformers | Streamlit

A machine learning application that evaluates how well a resume matches a job description using semantic similarity and structured skill extraction.

#### Key Features:
- Text preprocessing using spaCy (lemmatization, normalization, stopword removal)
- Skill extraction from unstructured text
- Transformer-based semantic similarity scoring (SentenceTransformers)
- Interactive Streamlit UI
- PDF resume parsing

#### What I Focused On:
- Clean modular architecture
- Real-world NLP pipeline design
- Handling noisy text data (C++, C#, .NET normalization)
- Making ML results interpretable
